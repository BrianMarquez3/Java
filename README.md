###  Tipos de Datos Primitivos en Java   

1. `byte`

  Representa un tipo de dato de 8 bits con signo. De tal manera que puede almacenar los valores num√©ricos de -128 a 127.
  
---
2. `short`

  Representa un tipo de dato de 16 bits con signo. De esta manera almacena valores num√©ricos de -32.768 a 32.767.

---
3. `int` 

  Es un tipo de dato de 32 bits con signo para almacenar valores num√©ricos. Cuyo valor m√≠nimo es -2^31 y el valor m√°ximo 2^31-1.

4. `float`

  Es un tipo dato para almacenar n√∫meros en coma flotante con precisi√≥n simple de 32 bits.

---
5. `double` 

  Es un tipo de dato para almacenar n√∫meros en coma flotante con doble precisi√≥n de 64 bits.

---
6. `boolean`

  Sirve para definir tipos de datos booleanos. Es decir, aquellos que tienen un valor de true o false. Ocupa 1 bit de informaci√≥n.

---
7. `char` 

  Es un tipo de datos que representa a un car√°cter Unicode sencillo de 16 bits.
